# Submit Graphite PR

This is an alias for `/graphite:submit`. Follow the same execution steps.

See the full documentation in the graphite plugin.

## Quick Reference

1. Validate environment (git repo + gt CLI)
2. Run `gt submit`
3. Report PR URL

Common errors: auth issues (run `gt auth`), network problems.
